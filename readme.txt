PacketPass is the working prototype of firewall manager that makes management of cross-geography security zones in the cloud easy and intuitive.

Objectives:
	Dynamic Security Zones with editable policies in real-time
	Add / Remove and move instances across security zones using drag/drop
	Define new groups
	Heterogeneity in end-point clouds
	Multi-Platform endpoints

Significance:
	Cloud environment different from conventional data center
		Less degree of control
		Network is a precious resource
	Business aspects
		Groups may represent departments (forking possible)
		Policies may be updated as architecture changes
